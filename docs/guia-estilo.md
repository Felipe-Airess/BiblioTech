# Guia de Estilos - BiblioTech

## 1. Nossas Cores (Paleta)

Escolhemos cores que lembram um ambiente de estudo e leitura, mas com tons de alerta para a parte de gestão e multas:

* **Azul Marinho (`#1E3A8A`) - Cor Primária:**
  * **Onde vamos usar:** No topo do site (cabeçalho) e nos botões principais, como "Confirmar Empréstimo".

* **Âmbar/Dourado (`#F59E0B`) - Cor de Destaque:**
  * **Onde vamos usar:** Botões secundários e etiquetas (ex: avisar que um livro é "Bestseller").

* **Vermelho (`#EF4444`) - Cor de Alerta:**
  * **Onde vamos usar:** Somente para coisas críticas, como avisar sobre multas, atrasos ou botão de "Excluir".

* **Cinza Claro (`#F3F4F6`) - Cor de Fundo:**
  * **Onde vamos usar:** No fundo de todas as telas do sistema.
  * **Por que escolhemos:** O branco puro cansa a vista muito rápido. Esse cinza bem clarinho deixa a leitura mais confortável.

* **Cinza Escuro (`#1F2937`) - Cor dos Textos:**
  * **Onde vamos usar:** Nos parágrafos, listas de livros e tabelas. Fica mais suave para ler do que o preto absoluto.

## 2. Fontes (Tipografia)

* **Inter (Sem serifa) - Fonte Base:**
  * **Onde vamos usar:** Menus, botões, tabelas de dados e formulários.

* **Merriweather (Com serifa) - Fonte dos Títulos:**
  * **Por que escolhemos:** Ela tem aqueles "puxadinhos" nas letras que lembram livros impressos tradicionais.
